# Jesty Versions

## Branches

### `master` (v1.0)
**Stable free version**
- Popup-based roasts (click extension icon)
- "Feed Jesty" support page
- Basic tab reading and roasting

---

### `premium-experiments` (v1.1)
**White Google screen + Jesty widget**
- Replaces Chrome's new tab with custom page
- Google logo + search bar (same layout as native)
- Jesty card with auto-roast on new tab
- Animated character expressions based on mood
- Side panel chat ("Talk back" feature)
- Icon buttons for refresh and chat
- Randomized roast selection with shuffled tabs
- Multiple tones: sarcastic, funny, assertive, savage, naughty, wholesome

---

### `experiment-hero-jesty` (v1.2)
**Hero Jesty - character takes center stage**
- White new tab screen
- Large, prominent character with expressive animations
- Quote as the main focus
- Minimal Google search bar
- Gmail + Google links in top right
- Character-first design: Jesty is the hero, not a widget
